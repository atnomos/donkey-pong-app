A retro arcade Pong game with a Donkey Kong theme. Built as a single-file HTML5 game.

## Features
- Arcade-style Pong gameplay vs CPU (Easy / Med / Hard)
- Choose your player: Plumber or Princess
- Neon glow effects on paddles and ball with CRT trail
- Vertical mobile mode (portrait orientation)
- Retro pixel font (Press Start 2P)
- localStorage leaderboard with initials entry (top 10)
- Original soundtrack by Diego
- Buy Me a Coffee tip button
- Google Analytics (GA4)

## How to Play
Open `index.html` in a browser. Click or tap to start music, then hit **PRESS START**.

- **W / S** or **Arrow Keys** ‚Äî move paddle
- **Space** ‚Äî start / pause
- First to **10 points** wins
